# zshrc

zshrc and powerlevel10k configration

## install zplug

```bash
cd ~
git clone git@github.com:zplug/zplug.git
```

## clone this repo and copy to $HOME

```bash
cp .zshrc ~/
cp .p10k.cfg ~/
```

maybe you should install Nerd Font if you use `powerlevel10k` plugin.

## install zsh plugins

```bash
zplug install
```

That's all.
